What is the map() function?

The map() function is used to transform every element in an array and create a new array with the transformed values. It doesn't change the original array - it returns a brand new array.

Key Points:

  • Takes each element from the original array
  • Applies a function to transform it
  • Returns a new array with the same length
  • Original array stays unchanged
