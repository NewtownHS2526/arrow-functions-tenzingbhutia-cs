What is the filter() function?

The filter() function is used to create a new array with only the elements that pass a test (condition). It filters out elements that don't meet the criteria, keeping only the ones that do.

Key Points:

  • Tests each element against a condition
  • Returns a NEW array (doesn't change the original)
  • Only includes elements where the condition is true
  • The new array can be shorter than the original
